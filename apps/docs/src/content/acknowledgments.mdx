# Acknowledgments

This project was made possible thanks to the generosity, creativity, and hard work of the open-source community. I'm deeply grateful to the developers, designers, and maintainers whose projects and ideas shaped and supported this work.

## Foundational Project

A special thanks to [@codewithsadee](https://github.com/codewithsadee) for the outstanding [vcard-personal-portfolio](https://github.com/codewithsadee/vcard-personal-portfolio). It served as the starting point and structural foundation for this site.

## Open Source Projects Used or Adapted

This project incorporates and adapts the following open-source libraries and tools:

### Core Libraries and Tools

* [**rehype**](https://github.com/rehypejs/rehype) — for transforming and processing HTML with plugins.
* [**remark**](https://github.com/remarkjs/remark) — for Markdown processing in MDX content.
* [**Giscus**](https://giscus.app/) — for enabling GitHub Discussions-based comments.
* [**react-scroll-motion**](https://github.com/1000ship/react-scroll-motion) — for scroll-based animations and transitions.
* [**GitHub Calendar**](https://github.com/grubersjoe/react-github-calendar) — for displaying contribution activity in a visual calendar format.
* [Rehype Pretty Code](https://rehype-pretty.pages.dev/) — for enhancing code block styling.
* [Shiki](https://shiki.style/) — for syntax highlighting in code blocks.
