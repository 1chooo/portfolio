## Features

- **Modern Stack Setup** ‚Äì Fast, responsive design powered by Next.js, Tailwind CSS, and TypeScript (strict mode), managed with Turborepo.
- **Content & Publishing** ‚Äì Supports Markdown/MDX, with RSS feed, sitemap generation, comment system, reading time display, and view counter.
